# MERawi Pharmacy Pro 🏥

A professional, offline-first pharmacy management system designed for Ethiopian pharmacies. Complete inventory management, sales tracking, business intelligence, and reporting tools.

---

## 📋 Features

### 💊 Inventory Management
- Complete medicine database with batch tracking
- Expiry date monitoring with color-coded alerts
- Low stock notifications
- Category-wise organization
- Location tracking (shelf/rack)

### 💰 Point of Sale
- Quick medicine search
- VAT calculation (configurable 0-15%)
- Professional receipts
- PDF receipt generation
- Print support
- Transaction history

### 📊 Business Intelligence
- Real-time dashboard with KPIs
- Inventory value tracking
- Profit margin analysis
- Sales performance metrics
- Category distribution charts
- Expiry status visualization
- Turnover rate calculation
- Days of inventory analysis

### 📈 Reports
- Low stock report
- Expiring medicines report
- Today's sales report
- Complete inventory report
- Excel export capability

### 🔒 Security & Data
- PC-specific license activation
- Automatic data backup
- Data stored in AppData (safe from updates)
- Excel import/export
- Receipts archive

---

## 💻 System Requirements

- **Operating System**: Windows 7/8/10/11, macOS, Linux
- **Python**: 3.8 or higher
- **Disk Space**: 100MB minimum
- **RAM**: 2GB minimum (4GB recommended)
- **Internet**: Not required (offline operation)
